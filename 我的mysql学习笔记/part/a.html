MySQL基本操作

数据库登录：

	mysql -u root -p
	Enter password: *********

修改密码：

	alter user 'root' @ 'localhost' identified by "new_password";

修改表名：

	rename table 表名 to 新表名;

修改列名：

	alter table 表名 change 列名 新列名 数值类型;

显示数据库与选择数据库：

	show databases；
	use databaseName;

退出数据库：

	exit；

创建数据库：

	create database test；

查看数据库中的表：

	show tables;

创建数据表

例：创建数据库test，并在test数据库中创建pet表，并在表中增加数据

创建数据库test：
	mysql> create database test;
	Query OK, 1 row affected (0.55 sec)

	mysql> use test
	Database changed
	mysql> CREATE TABLE pet(
		->         name VARCHAR(20),
		->         owner VARCHAR(20),
		->         species VARCHAR(20),
		->         sex CHAR(1),
		->         birth DATE,
		->         death DATE);
	Query OK, 0 rows affected (2.19 sec)

	mysql> insert into pet values('Puffball','Diane','hamster','公','1999-03-30',NULL);
	Query OK, 1 row affected (0.69 sec)	

查看数据表结构：
	mysql> describe pet;
	+---------+-------------+------+-----+---------+-------+
	| Field   | Type        | Null | Key | Default | Extra |
	+---------+-------------+------+-----+---------+-------+
	| name    | varchar(20) | YES  |     | NULL    |       |
	| owner   | varchar(20) | YES  |     | NULL    |       |
	| species | varchar(20) | YES  |     | NULL    |       |
	| sex     | char(1)     | YES  |     | NULL    |       |
	| birth   | date        | YES  |     | NULL    |       |
	| death   | date        | YES  |     | NULL    |       |
	+---------+-------------+------+-----+---------+-------+
	6 rows in set (0.22 sec)

查看表中的记录：	
	mysql> select * from pet;
	+----------+-------+---------+------+------------+-------+
	| name     | owner | species | sex  | birth      | death |
	+----------+-------+---------+------+------------+-------+
	| Puffball | Diane | hamster | 公   | 1999-03-30 | NULL  |
	+----------+-------+---------+------+------------+-------+
	1 row in set (0.00 sec)
	
修改数据：
	mysql>update pet set name='旺旺' where owner='Diane';
	mysql> select * from pet;                           
	+--------+-------+---------+------+------------+-------+
	| name   | owner | species | sex  | birth      | death |
	+--------+-------+---------+------+------------+-------+
	| 旺旺   | Diane | hamster | 公    | 1999-03-30 | NULL  |
	+--------+-------+---------+------+------------+-------+
	1 row in set (0.00 sec)
	
删除数据：
	mysql>delete from pet where name='旺旺';
